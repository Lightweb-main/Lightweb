Composite Functions

Introduction

Composite Functions in mathematics are essentially just a function inside another function, there isn't a simpler way to put it. For example, lets make two functions, f(x) and g(x)...

f(x) g(x)

And then give them their formulas...

f(x) = 2x + 3

g(x) = x - 5

And finally lets define an h(x) function...

h(x)

So in this example, h(x) is a composite functionh(x) = f(g(x))

So h(x) is the composite function, f(x) as the outer function, and g(x) as the inner function.

g(x) is essentially the x part in f(x) since it replaces x in f(x)...

f(g(x)) ---> x = g(x)

Evaluating Composite Functions

Evaluating composite function is finding the output of it. For example, lets give f(x) and g(x) equations...

f(x) = 2x - 3

g(x) = 3x + 1

And lets input 5 in g(x)...

g(5) = 3x + 1

Lets evaluate g(5)...

g(5) = 3(5) + 1

g(5) = 15 + 1

g(5) = 16

So g(5) equals 16, now lets input that into f(x)...

f(16) = 2x - 3

f(16) = 2(16) - 3

f(16) = 32 - 3

f(16) = 29

So finally f(g(5)) equals 29...

f(g(5)) = 29 --> g(5) = 16
