Composite Functions

Introduction

Composite Functions in mathematics are essentially just a function inside another function, there isn't a simpler way to put it. For example, lets make two functions, f(x) and g(x)...

f(x) g(x)

And then give them their formulas...

f(x) = 2x + 3 g(x) = x - 5

And finally lets define an h(x) function...

h(x)

So in this example, h(x) is a composite function, where g(x) is inside of f(x)...

h(x) = f(g(x))

So h(x) is the composite function, f(x) as the outer function, and g(x) as the inner function.

g(x) is essentially the x part in f(x) since it replaces x in f(x)...

f(g(x)) ---> x = g(x)

Evaluating Composite Functions

Evaluating composite function is finding the output of it. For example, lets give f(x) and g(x) equations...

f(x) = 2x - 3 g(x) = 3x + 1

And lets input 5 in g(x)...

g(5) = 3x + 1

Lets evaluate g(5)...

g(5) = 3(5) + 1 g(5) = 15 + 1 g(5) = 16

So g(5) equals 16, now lets input that into f(x)...

f(16) = 2x - 3 f(16) = 2(16) - 3 f(16) = 32 - 3 f(16) = 29

So finally f(g(5)) equals 29...

f(g(5)) = 29 --> g(5) = 16

Finding Composite Functions

To find a composite function, substitute x in the outer function with g(x), the inner function. For example...

f(x) = 2x2 + 3x g(x) = 4x + 4 Whats f(g(4))?

Lets substitute the x in f(x) with 4x + 4...

f(4x + 4) = 2(4x + 4)2 + 3(4x + 4) f(4x + 4) = 2(16x2 + 32x + 16) + 12x + 12 f(4x + 4) = 32x2 + 64x + 32 + 12x + 12 f(4x + 4) = 32x2 + 76x + 44

So the composite function of g(x) and f(x) is 32x2 + 76x + 44.

Commutative Property Doesn't Apply

The commutative property in multiplication where the order of multiplication results in the same answer does not apply to composite functions, changing the order of composing will result in a different answer...

f(x) = 2x - 1 g(x) = x2 What's f(g(3)) g(3) = 32 g(3) = 9 f(9) = 18 - 1 f(9) = 17 What's g(f(3)) f(3) = 6 - 1 f(3) = 5 g(5) = 52 g(5) = 25

Notice how flipping the outer function with the inner function results in a different answer.

Notation of Composing Functions

Composing functions can be shown through a few ways, like through having the inner function inside the parantheses of the outer function...

f(g(x))

This means that g is composed in f.

They can also appear with the two function letters being multiplied together...

(f ⋅ g)(x)

The order of multiplication is very important as the second letter is the function that is composed by the other function...

(f ⋅ g)(x) = f(g(x)) (g ⋅ f)(x) = g(f(x))

The inner function can also be enclosed in sqaure brackets...

f[g(x)]

This is to show that the x in f(x) is being substituted by g(x).
