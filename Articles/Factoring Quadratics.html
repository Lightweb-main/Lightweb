Factoring Quadratics

Introduction

Factoring is breaking apart an equation into its factors that multiply together to make up the equation like this one…

x2 + x - 42 = 0 (x + 7)(x - 6)

To factor a quadratic equation, look at an equation like this one…

x2 + 3x - 18 = 0

To form a quadratic, two 1x values multiplied together is x2 however the coefficient values may vary the equation…

(x)(x) = x2

To get b and c in a quadratic, find two numbers that multiply to c but add up to b’s coefficient. In other words, two numbers must multiply to -18 but add up to 3…

(x + 6)(x - 3) = x2 + 3x - 18

The numbers +6 and -3 both add to 3 which is b, and multiply to -18 which is c. So x + 6 times x - 3 is x2 + 3x - 18 factored…

x2 + 3x - 18 = 0 = (x + 6)(x - 3)

Factorization isn’t to simplify the equation, it can be used to get the roots of the quadratic equation. To get a factored quadratic’s roots, get each of the factors and equal them to zero…

x + 6 = 0 x - 3 = 0

Solve them like linear equations…

x + 6 = 0 x = -6 x - 3 = 0 x = 3

The roots for x2 + 3x - 18 are -6 and 3.

Keep in mind that some quadratic equations cannot be factored as two numbers that add up to c and multiply two b are not possible. Always use the quadratic formula when factorization isn’t possible.

When a ≠ 1

When the constant a is not equal to 1, the process of factorization changes...

3x2 - 8x - 3 = 0

When a ≠ 1, the first step is to find a pair to split the variable b (-8 in this example) into two coefficents that when multiplied equals the product of a and c (a = 3, c = -3)...

let d = ac d = 3(-3) = -9 d = -9

Now that we've found the product of a and c, we now need to find two numbers that multiply to equal -9 and add to equal -8...

All Factors of -9 (1, -9) (3, -3) All Factors of -9: -9, -3, 1, 3 Sum of -8: -9 + 1

Since we've found our two numbers, we can split b into two parts...

3x2 - 8x - 3 = 0 3x2 - 9x + x - 3 = 0

Next, we split the quadratic itself into two parts, one part will have the first factor, and the second will have the other...

3x2 - 8x - 3 = 0 (3x2 - 9x) + (x - 3) = 0

Then, we factor out the GCF (Greatest Common Factor) of both terms. We'll use the GCFs of both terms then add them together in the next step...

3x(x - 3) + 1(x - 3) = 0

Lastly, we take the GCFs, add them up and then multiply the first term with it (while also removing the second term)...

(x - 3)(3x + 1) = 0

You can also check your answer by mutliplying the two terms and making sure the product equals the original quadratic...

(3x + 1)(x - 3) = 3x2 - 8x - 3 Applying the FOIL method: 3x(x) + 3x(-3) + 1(x) + 1(-3) = 3x2 - 8x - 3 3x2 - 9x + x - 3 = 3x2 - 8x - 3 Notice how the result of the FOIl method is the same equation as when we split up b into two parts 3x2 - 8x - 3 = 3x2 - 8x - 3 ✓

The results are the same, meaning that the factored form of the quadratic that we got is correct.

Steps for factoring (a ≠ 1)
