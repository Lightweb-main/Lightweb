The Steps To Derive The Quadratic Formula

To derive the quadratic formula, take the quadratic standard form...


a x2  +  bx  +  c  = 0


First, divide both sides by a...


x2  +  b a x  +  c a  = 0


Next, put c/a on the other side...


x2  +  b a x  =  - c a


Now we add (b/2a)2 to both sides...


x2 + ba x + ( b 2a 2 ) = ( b 2a 2 ) - ca


Now rewrite the terms on the left as this...


( x + b 2a 2 ) = ( b 2a 2 ) - ca


Now square root both sides...


x + b 2a = ( b 2a 2 ) - ca
