The Steps To Derive The Quadratic Formula

To derive the quadratic formula, take the quadratic standard form...


a x2  +  bx  +  c  = 0


First, divide both sides by a...


x2  +  b a x  +  c a  = 0


Next, put c/a on the other side...


x2  +  b a x  =  - c a


Now we add (b/2a)2 to both sides...


x2 + ba x + ( b 2a 2 ) = ( b 2a 2 ) - ca


Now rewrite the terms on the left as this...


( x + b 2a 2 ) = ( b 2a 2 ) - ca


Now square root both sides...


x + b 2a = ± ( b 2a 2 ) - ca


Next, move the +b/2a to the right...


x + = b 2a ± ( b 2a 2 ) - ca


Now that x is on it's own, we've derived the quadratic formula, but we can simplify it...


First, multiply the right by 2a/2a...


x = - b ± -  c a * ( 2a )² + ( b 2a )²  * ( 2a )²  2 ⁢ a


This formula can be simplified to...


x = - b ± b 2 - 4 ⁢ a ⁢ c 2 ⁢ a
