Angular Velocity

Introduction

Angular Velocity quantifies how fast an object is moving around an axis, or a center point. Similar to the formula for regular velocity, angular velocity is angular displacement over time...

Angular Velocity  =  Angular Displacement Time

But in physics, the formula for angular velocity looks different...

ω  =  Δθ t

So the weird w at the start of the equation is the lowercase version of the greek letter, omega which represents angular velocity. Also the triangle on top of the fraction is an uppercase delta which represents the differece between two values, in which said value is a lowercase theta which represents an angle. Lastly the t at the bottom represents time.
