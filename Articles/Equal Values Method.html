Equal Values Method

Introduction

The Equal Values Method is a common method to solve systems of equations by making two expression equal to each other and solving the remaining equation.

Examples

Example #1

Here's an example...

5x + 3 x - 5

These two expressions are what makes up a system of equations and they can either have one, none, or even infinite solutions.

To start, first make the two expressions equal to each other...

5x + 3 = x - 5

Next, in this equation, we subtract x on both sides...

4x + 3 = -5

Then, we subtract 3 on both sides...

4x = -8

Lastly, we divide both sides by 4 to isolate x...

x = -2

So by using the equal values method we got the value of x which is -2...

Example #2

Here's another example...

3x - 1 2x + 2 Same as the last example, we first make the both expressions equal to each other... 3x - 1 = 2x + 2 Then we find x by isolating it... 3x - 1 = 2x + 2 x - 1 = 2 x = 3 By using the Equal Values Method, you get 3 as the value of x...
