Absolute Value of Numbers

Introduction

The absolute value of numbers is their distance away from zero. Basically the absolute value of a number is itself as a positive number…

|2| = 2 |-3| = 3 |6| = 6 |-12| = 12 |-73| = 73

The absolute value of a number is always positive with only one exception…

- |2| = -2 - |-7| = -7 - |4| = -4 - |-11| = -11 - |-89| = -89

This example has a negative sign, which means that no matter what integer you put, you will always get a negative number.

An absolute value cannot equal a negative number without a negative sign like this example…

|x| = -2

This example doesn’t have a solution since no matter what value x is, it can’t equal a negative number since its distance from zero cannot be negative therefore this equation has no solution…

|x| = -2 NO SOLUTION
