Power Rule in Calculus

Introduction

The power rule in calculus is a formula used to find the derivatives of a variable exponentiated. The power rule can be used to find derivatives and the antiderivatives have different formulas…

d/dx= nxn - 1 ∫xndx= xn+1/ n+1

Derivatives

An example of the power rule in derivatives would be…

What is d/dx(x3 + x)? d/dx x3 + 1x

In differential calculus, the power rule is very simple. You multiply the coefficient by the exponent…

x3 + 1x —> 3x3 + 1x

After that the power of each base is subtracted by 1…

3x2 + x0 = 3x2 + 1 d/dx x3 + 1x = 3x2 + 1

Another example would be…

d/dx(2x3 + x2 + 3)

The first step would be to multiply all the coefficients with their respective powers…

6x3 + 2x2 + 3

Then we find the derivatives of all the values inside the parentheses…

d/dx(2x3) = 6x2 d/dx(x2) = 2x d/dx(3) = 0 The derivative is 6x2 + 2x

Antiderivatives

The power rule can also be used on antiderivatives, however the formula is very different…

∫xn dx = xn+1 / n+1

To get the antiderivative of an equation, you would have to increase the power by 1 and divide it by the same sum. For example…

∫12x6dx

First we increase the power by 1…

12x7

Then we divide it by 7 since the new power is 7…

12x7 / 7

The antiderivative of 12x6 is 12x7 divided by 7 + C…

∫12x6dx = 12x7 / 7 + C
