Derivative Rules & Formulas

Introduction

Derivative Rules are a set of formulas used to find the derivative of functions. The rule used to find the derivative solely depends on the function itself.

Derivative Rules

Power Rule

The power rule is a common way to find the derivative of a function. An example of the power rule being used is…

d/dx 8x5 = 40x4

The formula for the power rule is…

d/dx xn = nxn - 1

Basically the formula is to multiply the coefficient of the variable with the exponent before subtracting it by 1…

d/dx 2x3 = 6x2

d/dx 5x6 = 30x5

d/dx 9x3 = 27x2

d/dx 2x7 = 14x6

d/dx 9x6 = 54x5

This can also work with rooted variables…

d/dx 5√x

5√x = x1/5

d/dx 5√x = 1/5x-4/5

The power rule can only work on variables that have exponents that are not variables themself. These examples are incompatible with the power rule…

d/dx 2x

d/dx ex

d/dx 5x2x

d/dx 1/6xx/3

You'll see that the power rule will be used a lot. Other rules often use the power rule.

Constant Derivative Rule

The Constant Derivative Rule is a formula that every constant in a derived function is always zero…

f’(c) = 0

The rule can only be applied if c is a constant. This rule can be used to shorten derivatives like…

f(x) = -2 f’(x) = 0

g(x) = 5 g’(x) = 0

t(x) = 8 t’(x) = 0

a(x) = 3 a’(x) = 0

This can also be used with derivatives with multiple numerals…

d/dx(2x3 + 5x2 + 5x + 3)

= 6x2 + 5x + 5 + 0

= 6x2 + 5x + 5

Constant Multiple Rule

The Constant Multiple Rule is a formula to find the derivatives of a function multiplied by a constant…

d/dx(C * f(x) = C * f’(x)

The Constant Multiplier Rule states that the derivative of a function multiplied by a constant is the derived function multiplied by the same constant…

d/dx(4x7)

= (4)(7x6)

= 28x6



d/dx(7x5)

= (7)(5x4)

= 35x4

The Constant Multiplier rule is also another formula for the power rule when there is a coefficient with the variable. This rule can also be used on fractions with variables like…

d/dx(x/5)

= (⅕ * x0)

= 1/5



d/dx(3x/8)

= (⅜ * x0)

= 3/8



d/dx(2/6 * x2)

= (4/6 * x)

= ⅔ * x

= 2x / 3



d/dx(x3/7)

= (1/7 * 3x2)

= 3x2/7



d/dx(-3 4√x)

= (-3)(x1/4)

= (-3)(x-3/4)

= -3x-3/4

The Constant Multiple Rule can be used to find the derivatives of fractions with exponents that are harder to do with the power rule.

Sum Rule

The Sum Rule is a derivative formula used to find the sum of multiple derived functions…

d/dx(f(x) + g(x)) = ???

The Sum Rule states that the derivative of the sum of multiple functions is equal to the sum of those functions derived.

d/dx(f(x) + g(x)) = f’(x) + g’(x)

For example…

d/dx((2x4 + 3x2 + 2x) + (5x3 + x2 + 6x))

= (8x3 + 6x + 2) + (15x2 + 2x + 6)

= 8x3 + 15x2 + 8x + 8

To do the sum rule on derivatives, first derive every derivative of every term so…

d/dx(2x4) = 8x3 d/dx(3x2) = 6x d/dx(2x) = 2

So the derivative of the first function is 8x3 + 6x + 2. Now find the derivative of the second function…

d/dx(5x3) = 15x2 d/dx(x2) = 2x d/dx(6x) = 6

The derivative of the second function is 15x2 + 2x + 6. Now add them together…

68x3 + 6x 15x2 + 2x + 6 8x3 + 15x2 + 8x + 6

The final answer of both of the derived functions equal 8x3 + 15x2 + 8x + 6…

d/dx((2x4 + 3x2 + 2x) + (5x3 + x2 + 6x)) = 8x3 + 15x2 + 8x + 6

Now here are more examples…

d/dx((7x5 - 12x3 - 4x2) + (-3x6 + 2x2 + 8x))

d/dx(7x5 - 12x3 - 4x2) = 35x4 - 36x2 - 8x

d/dx(-3x6 + 2x2 + 8x) = -18x5 + 4x + 8

= 35x4 - 36x2 - 8x - 18x5 + 4x + 8

= -18x5 + 35x4 - 36x2 - 4x + 8

Difference Rule

The difference rule is a formula used to find the difference between two derived functions…

d/dx(f(x) - g(x))

The difference rule is basically the reverse of the sum rule in which the difference of the derivatives of multiple functions is equal to every derived function all subtracted from each other…

d/dx(f(x) - g(x)) = f’(x) - g’(x)

An example of the difference rule would be…

d/dx(2x4 - 5x3)

= 8x3 - 15x2

Just like the sum rule, find the derivative of both terms before subtracting…

d/dx(2x4 - 5x3) d/dx(2x4) = 8x3 d/dx(-5x3) = (-5 * 3x2) = -15x2

= 8x3 - 15x2

Product Rule

The Product rule is a formula used to find the product of two derivatives…

d/dx(f(x) * g(x)) = f’(x) g + f g’(x)

An example of this would be…

d/dx((2x3 - 12x2 + 4)(2x - 3)) = ???

Same as before, find the derivative of the first function…

Quotient Rule

The Quotient rule is used to find the derivative of the quotient of two functions…

f(x) = g(x) / h(x) f(x) = g' h - h' g / h2

Basically the quotient rule is the product of the derived first function and the second function minus the derived, second function with the first function over the second function squared. If that sounds confusing, an example would be…

f(x) = (6x6 - 4x5 + 8x4) / (3x2 + 10x) f’(x) = ???

First we find the product of the derivative of the first function with the second function…

(36x5 - 20x4 + 32x3) * (3x2 + 10x)

= (36x5 - 20x4 + 32x3) * (3x2) + (36x5 - 20x4 + 32x3) * (10x)

= (108x7- 60x6 + 96x5) + (360x6 - 200x5 + 320x4)

= 108x7 + 300x6 - 104x5 + 320x4

Now we find the product of the derivative of the second function with the first function…

(6x6 - 4x5 + 8x4) * (6x + 10)

= (6x6 - 4x5 + 8x4) * (6x) + (6x6 - 4x5 + 8x4)(10)

= (36x7 - 24x6 + 48x5) + (60x6 - 40x5 + 80x4)

= 36x7 - 36x6 + 8x5 + 80x4

Now we subtract the two products…

108x7 + 300x6 - 104x5 + 320x4

36x7 - 36x6 + 8x5 + 80x4

72x7 + 264x6 - 112x5 + 240x4

Then we find the second function squared so we can get our denominator…

(3x2 + 10x)2

= (3x2 + 10x)(3x2) + (3x2 + 10x)(10x)

= (9x4 + 30x3) + (30x3 + 100x2)

= 9x4 + 60x3 + 100x2

The final answer is 72x7 + 264x6 - 112x5 + 240x4 over 9x4 + 60x3 + 100x2…

f’(x) = 72x7 + 264x6- 112x5 + 240x4 / 9x4+ 60x3 + 100x2

Chain Rule

The Chain rule is a formula used to find the derivative of a function of a function of a function…

f(g(x)) = f'(g(x))g'

f * g = (f * g) * g'

dy * dx = dy / du * du / dx

An example of the chain rule would be a function in a function…

d/dx(2x4 - 3x3)4

f(g) = g4

g(x) = (2x4 - 3x3)

First we find the product of the first derived function with the second function…

d/dx(2x4 - 3x3) = 8x3 - 9x2

d/dx(g4) = 4g3

(4g(x)3)(8x3 - 9x2) =

4(g(x)3(8x3 - 9x2)

The final answer is 4(g(x)3) times (8x3 - 9x2)...

d/dx(2x4 - 3x3)4 = 4(g(x)3(8x3 - 9x2)

Derivative of Constants

The derivative of any constant is always zero, any single constant, negative and positive…

f(x) = 4
f’(x) = 0
g(x) = -3
g’(x) = 0



h(x) = 7
h’(x) = 0
i(x) = -10
i’(x) = 0

f’(x) = 0 x = -∞ to ∞

Derivative of Terms to the Power of 1

Any term that has a variable to the power of 1 will have their derivative without it…

f(x) = 7x
f’(x) = 0

g(x) = -2x
g’(x) = 0

h(x) = -12x
h’(x) = 0

i(x) = 34x
i’(x) = 0

f’(ax) = a
a = -∞ to ∞

Derivative of Terms with Exponents Greater Than 1

To find the derivative of terms with exponents greater than 1, apply the power rule in which multiply the coefficient by the power then subtract the exponent by 1…

f(x) = 7x3
f’(x) = 21x2

g(x) = 10x4
g’(x) = 40x3

h(x) = 2x10
h’(x) = 20x9

i(x) = x2
i’(x) = 2x

f’(xn) = nxn - 1
f’(axn) = a(nxn - 1)
a = -∞ to ∞

Derivative of Terms with Exponents Less Than 1

The derivative of terms with exponents less than 1 can also be applied to the power rule…

f(x) = 3x
f’(x) = 1.5x-1/2

g(x) = 6√-4x
g’(x) = -4/6x-5/6

h(x) = 2x-3

Derivative of Exponentials

The derivatives of exponentials are dependent on the base. If it is e (Euler’s Number) then it doesn’t change, but if it is a coefficient, it will change into the natural logarithm multiplied by it to the power of x…

f(x) = ex
f’(x) = ex

g(x) = 5x
g’(x) = ln(5) * 5x OR 5x ln(5)

h(x) = -12x
h’(x) = ln(-12) * -12x OR -12x ln(-12)

i(x) = 5x - 3
i’(x) = ln(5) * 5x - 3 OR 5x - 3 ln(5)

f’(ax) = ln(a) * ax

Derivative of Logarithms

The derivative of natural logarithms is 1 over the value in the logarithm. However the derivative may change when it is a normal logarithm…

f(x) = log3(5)
f’(x) = 1 / (5 ln(3))

g(x) = log7(-12)
g’(x) = 1 / (7 ln(-12))

h(x) = ln(3)
h’(x) = 1 / 3

h(x) = ln(x4)
h’(x) = 1/x4 d/dx(x4) = 4x3 / x4 = 4/x

f’(ln(xn) = n / x f’(loga(b)) = 1 / (b ln(a)

Derivative of Trig Functions

The derivatives of trig functions are not consistent, unlike powers of logarithms, trig functions may change with exponents involved…

Three Main Trig Functions

d/dx (sin(x)) = -cos(x)
d/dx (cos(x)) = sin(x)
d/dx (tan(x)) = sec2(x)


Reciprocals

d/dx(sec(x)) = sec(x)tan(x)
d/dx(csc(x)) = -cot(x) * csc(x)
d/dx(cot(x)) = -csc2(x)


Inverse Trig Functions

d/dx(arcsin(x)) = 1 / √(1 - x2)
d/dx(arccos(x)) = -1 / √(1 - x2)
d/dx(arctan(x)) = 1 / √(1 + x2)
