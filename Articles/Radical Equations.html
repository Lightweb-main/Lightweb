Radical Equations

Introduction

Radical equations are mathematical equations that contain a variable or known value in a radical symbol (√). Here are some examples...

x + 3 = 4

4x - 1 = -2x - 3

x = 3 4 3

Solving Radical Equations

When the Equation is

√ x = a

When the radical equation is √x equal to a constant ("a" as a placeholder), you raise both terms by the index of the root. So for example...

2x + 3 = 10 To solve this equation, you multiply both sides by themselves two times since the index of the radical is 2...

( 2x + 3 ) 2 = (10) 2

By doing this, we remove the radical or in this example the square root on the left...

2x + 3 = 100 Now we solve it as a linear equation...

2x + 3 = 100

2x = 97

x = 48.5

Make sure to check your answer by substituing x as it's supposed value to avoid potential extraneous solutions or any numbers that seem like solutions but aren't...

2x + 3 = 10

2(48.5) + 3 = 10

97 + 3 = 10

100 = 10

10 = 10   ✓

Since 48.5 satifies the equation, it is considered a solution...

x = 48.5

Example #2

Here's another example... 4x - 2 3 = 7

This is similar to the last example, except that the expression on the left is being rooted to the 3rd root, or in other words the index of the radical is 3. In order to solve this equation, you instead multiply both sides three times of themselves...

( 4x - 2 3 ) 3 = ( 7 ) 3

4x - 2 = 343

4x = 345

x = 86.25

Now to check if 86.25 is a solution or not...

4(86.25) - 2 3 = 7

345 - 2 3 = 7

343 3 = 7

7 = 7   ✓

86.25 satifies the equation, so it's a solution...

x = 86.25

Solving Radical Equations with variables on both sides.

Radical equation can also have variables on both sides like this one...

5x - 6 = 2x - 2

In order to solve this, you do the same thing and raise both sides depending on the value of the index so if the index is 2, you raise both sides to the second power but unlike the previous examples, you will get a polynomial on one or both sides, use different formulas like the quadratic formula if the polynomial is a quadratic or others to get the value of x...

( 5x - 6 ) 2 = ( 2x - 2 ) 2

5x - 6 = 4x 2 - 8x + 4
