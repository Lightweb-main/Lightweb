Order of Derivatives

Introduction

Order of derivatives is the repeating process of differentiation…

f’(x) = 4x3 f’’(x) = 12x2 f’’’(x) = 24x

Higher order derivatives are functions that have been derived multiple times, as in the example above, the functions first derivative is 4x3, the functions second derivative is 12x2, and the functions third derivative is 24x. The derivatives are noted in order like the first, second, third, and forth derivative. The function itself will have multiple apostrophes to show that it is being derived multiple times…

f(x) = 3x5 = 3x5 - 0.5 = 3x4.5 f’(x) = 13.5x3.5 f’’(x) = 47.25x2.5 f’’’(x) = 118.125x1.5 f’’’’(x) = 177.1875x0.5

The amount of apostrophes is the amount the function is derived…

f’’(x) = Second Derivative f’’’(x) = Third Derivative f’(x) = First Derivative
