Antiderivatives

Introduction

Antiderivatives (also called, reverse derivatives, primitive integrals, and primitive functions) are calculus equations that are the complete opposite of derivatives hence the name.

There are differences and similarities between antiderivatives and derivatives. First antiderivatives come from integrals which have integral signs whereas derivatives come from notation…

d d x

Antiderivatives:   ∫ 3 x  d x Derivatives:   d d x  3 x

Also the antiderivatives have a constant added…

Antiderivatives:   ∫ 3 x  dx = 3 x 2 2 = 1.5 x 2 + C Derivatives:   d d x  3 x = 6

Solving Antiderivatives

To find antiderivatives, you simply do the opposite for derivatives. For example the power rule turns from this…

d d x x n = n x n - 1

To this…

∫ x n   d x = x n + 1 n + 1

For example…

∫ 3 x 4 d x

First we add one to the exponent…

∫ 3 x 4 d x = 3 x 4 + 1 = 3 x 5

Then we divide by the new exponent...

3 x 5 5

The antiderivative of 3x4 is...

∫ 3 x 4 d x = 3 x 5 5

Antiderivatives vs Integrals

Antiderivatives are also associated with indefinite integrals as they both reverse the process of differentiation. But the difference between them is that antiderivatives of functions are also functions, and an indefinite integral is a set of all the antiderivatives...

Indefinite Integral:   ∫ 3 x 4 dx Antiderivatives:   3 x 5 5 + C

Antiderivative Notation

Antiderivatives are shown in numerous ways. Antiderivatives are shown as a capital F…

f ( x ) = 3 x 5 F ( x ) = x 6 2
