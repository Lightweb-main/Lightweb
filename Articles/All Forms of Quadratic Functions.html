Introduction

This is a quadratic...

2x2 + 3x + 2 = 0

A quadratic is an equation in which the highest power is 2, a quadratic on a graph shows the shape of a parabola or like an unclosed oval... A quadratic, much like a linear equation can be expressed in many different forms, the most common being standard form which is the typical...

ax2 + bx + c = 0

This is the form that quadratics are typically written in, like these ones...

3x2 - 2x + 6 = 0

-x2 + 3x - 1 = 0

5x2 + x + 2 = 0

2x2 - 7x - 4 = 0

Standard form is particularly useful in finding the roots (the solutions) of a quadratic and the y-intercept (what y equals when x = 0), these can be found using either factoring...

x2 + 3x - 4 = 0


Since 4 + (-1) = 3 and 4 * -1 = -4

(x + 4)(x - 1) = 0

x + 4 = 0 and x - 1 = 0

x = -4 and x = 1

x = -4, 1

Or through, the quadratic formula...

x = -b ± b ² - 4 a c 2a

These two methods can be used to find the roots of a quadratic, so find the y-intercept of a quadratic, you subsitute 0 as x and change the zero on the right to a y...

3x2 - 2x + 3 = 0

3(0)2 - 2(0) + 3 = y

3 = y

So the y-intercept of this quadratic is 3.

Standard Form

The standard form of a quadratic is the most commonly way to write quadratics, it is composed of 4 terms, 3 terms all combined equal to 0...

ax2 + bx + c = 0

The standard form is the most useful for determining the y-intercept of a quadratic and is just the easiest to understand.

Factored Form

The factored form of a quadratic shows the constant, "a", together with the two linear expressions that multiply with a that form the quadratic...

f(x) = a(x - r1)(x - r2)

r1 and r2 are the roots of the quadratic. This form of quadratics is useful because you can easily tell what the roots of the quadratic are. For example...

x2 + 5x + 6 = 0

This quadratic unlike some others can be evenly factored. First we would have to find two numbers that add to "5" and multiply to equal "6", to do this, we can write every factor pair (two numbers that can multiply up to a certain numbers, called factors) of 6...

(1, 6)

(-1, -6)

(2, 3)

(-2, -3)

Now we need to find the factor pair that the sum of the two numbers equals 5...

(1 + 6) = 7 ≠ 5

(-1 - 6) = -7 ≠ -7

(2 + 3) = 5 = 5

(-2 + -3) = -5 ≠ 5

Since the factor pair, (2, 3) equals 5, that means that we can use those numbers to split up the coefficent of bx in the original quadratic like this...

x2 + 5x + 6 = 0

x2 + (2 + 3)x + 6 = 0

x2 + 2x + 3x + 6 = 0

Now that we've split b in the quadratic, we now seperate the two terms...

(x2 + 2x) + (3x + 6) = 0

Now that the quadratic is split into two, we can now factor the two seperate terms seperately...

x(x + 2) + 3(x + 2) = 0

Lastly, we factor out (x + 2) since both terms contain it...

x(x + 2) + 3(x + 2) = 0

(x + 3)(x + 2) = 0

Now we've got our answer, x2 + 5x + 6 = 0 in factored form is (x + 3)(x + 2) = 0.


We can now find the roots of this quadratic through the Zero Product Property which states...

If a or b = 0, ab = 0

With this property, this automatically makes it...

x + 3 = 0 and x + 2 = 0

Now, we just solve the linear equations to get the roots...

x + 3 = 0

x + 2 = 0



x = -3

x = -2

x = -3, -2

And so the roots of this quadratic is -3 and -2.

If a ≠ 1

But if a ≠ 1, we would have to do an additional step at the beginning. For example...

3x2 + 9x + 6 = 0

First we can see that all the terms are divisible by 3, which is what a equals, so we can factor 3 out of the quadratic...

3(x2 + 3x + 2) = 0

Then we factor the quadratic inside the parantheses...

3(x2 + 3x + 2) = 0

3(x2 + 3x + 2) = 0

Factors of 2:

(1, 2)

(-1, -2)

(2, 1)

(-2, -1)



1 + 2 = 3 = 3

-1 - 2 = -3 ≠ 3

2 + 1 = 3 = 3

-2 - 1 = -3 ≠ 3

3(x2 + x + 2x + 2) = 0

3((x2 + x) + (2x + 2)) = 0

3(x(x + 1) + 2(x + 1)) = 0

3(x + 2)(x + 1) = 0

And so, x2 + 9x + 6 is 3(x + 2)(x + 1).

If b And c Are Divisble By a

But for a quadratic which cannot be factored by a constant like the previous example has an alternate way of figuring out the factored form of it...

3x2 - x - 10 = 0

We cannot divide 1 by 3 or 10 by 3 as they turn into fractions and it would be much harder finding the factors of a fraction than an integer. So instead we'll multiply the coefficent of ax (a) and cx (c) together to get our number of which we can find which factor pair equals b (-1)...

3 * (-10) = -30

Factors of -30:

(-6, 5) → -6 + 5 = -1 = -1

(6, -5) → 6 - 5 = 1 ≠ -1

(-10, 3) → -10 + 3 = -7 ≠ -1

(10, -3) → 10 - 3 = 7 ≠ -1

(30, -1) → 30 - 1 = 29 ≠ -1

(-30, 1) → -30 + 1 = -29 ≠ -1

Since we've found our factor pair which multiplies to -30 and adds up to -1, we can then factor the quadratic normally...

3x2 - x - 10 = 0

3x2 -6x + 5x - 10 = 0

(3x2 - 6x) + 5(x - 2) = 0

3x(x - 2) + 5(x - 2) = 0

(3x + 5)(x - 2) = 0

So...

3x2 - x - 10 = 0 → (3x + 5)(x - 2) = 0

Quadratic Cannot Be Factored At All

If a quadratic cannot be factored at all, use the quadratic formula to get your roots, and for these quadratics, you have to factor out the coefficent of ax...

3x2 - 5x - 10 = 0

First we factor out a from the quadratic...

3(x2 - 5/3x - 10/3) = 0

Then we use the quadratic formula to find the roots which will then be plugged back into the quadratic...

x = 5 ± (-5) ² - 4 (3) (-10) 2(3)

x = 5 ± 25 + 120 6

x = 5 ± 145 6

r1 = 5 - 145 6

r2 = 5 + 145 6

Now that we've got our roots, we can plug them into the quadratic...

3(x -  5 - 145 6 ) ( x -  5 + 145 6 )

This is the quadratic in factored form.

Vertex Form

The vertex form of a quadratic is another form of how quadratics can be expressed. The vertex form contains the coordinates of the vertex of the quadratic which is the highest/lowest point of the quadratic (the vertex can either be the highest or the lowest depending on if a > 0 or a < 0)...

f(x) = a(x - h)2 + k

The vertex form is useful for finding the coordinates of the vertex, the axes of symmetry (the imaginary line that splits the parabola in two, equal parts), and makes it easier to graph.

Conveting Standard Form to Vertex Form

To convert a quadratic in standard form to vertex, you can Complete The Sqaure. Here's an example...

x2 + 2x + 2

First you would factor out a out of the quadratic, but we have 1 so we can skip this step. Next you take the coefficent b in the equation and half it...

x2 + 2x + 1

2 / 2 = 1

Next, we multiply the quotient by itself (sqaure or ^2)...

1 * 1 = 1

After that, we add and subtract it to the quadratic itself...

x2 + 2x + 2 + 1 - 1

Then, we convert ax2 + bx + 1 (from b / 2) in the original quadratic (x + h)2 where h is b / 2 or 1 from our quadratic...

x2 + 2x + 2 + 1 - 1

(x + 1)2 + 2 - 1

Lastly, we simplify by adding 2 and -1...

(x + 1)2 + 2 - 1

(x + 1)2 + 1

And here is the quadratic in vertex form. Here's a more challenging quadratic to convert into vertex form...

3x2 - 6x + 5

Now for this quadratic, we can't complete the sqaure yet since a is not equal to 1. So we would do an extra step and factor out a out of the quadratic (in this case 3)...

3x2 - 6x + 5

3(x2 - 2x + 5/3)

Now that a equals 1, we can complete the sqaure...

3(x2 - 2x + 5/3)


b = -2, b / 2 = -1


3(x2 - 2x + 5/3 + (-1)2 - (-1)2)


3(x2 - 2x + 5/3 + (-1)2 - (-1)2)


3((x - 1)2 + 5/3 - (-1)2)


3((x - 1)2 + 2/3)


Now we take c (or 2/3) out of the factoring...

3(x - 1)2 + 2

And now the quadratic is converted into vertex form.

Finding the vertex coordinates from vertex form

A Top/Bottom Parabola

You can find the vertex's coordinates from vertex form from h and k...

a(x - h)2 + k

The x-coordinate of the vertex is h...

(h, )

And the y-coordinate of the vertex is k...

(h, k)

So from the previous example...

3(x - 1)2 + 2

We can see that h = 1 and k = 2...

3(x - 1)2 + 2

h = 1

x - h = 0

x - 1 = 0

h = 1

k = 2

This means that the vertex is at (1, 2). And to find the axes of symmetry, the x value of it equals h...

x = h

So...

x = 1

A Left/Right Parabola

To find the vertex of a left/right opening parabola, you instead change the x in the vertex form to y, so...

x = (y - h)2 + k

The method of finding out the coordinates of a vertex in a horizontal parabola stays the same however...

(h, k)

So for example...

x = (2y - 4)2 + 3

We would find the values of h and k...

2h - 4 = 0

2h = 4

h = 2

k = 3

(2, 3)

So the coordinates of this horizontal parabola is (2, 3).

Summary

Glossary

Parabola: A u-shaped curve, formed by quadratics, it's shape / direction dependent on the quadratic

Roots:
The solutions to a polynomial; Two, even numbers that multiply up to a certain number.

Y-intercept: The value of y when x equals 0; Points that crosses the y-axis.

Factor Pair: Two numbers, grouped up and multiplied to equal a certain number. [1, 3] = 1 * 3 = 3

Factors: Two numbers that multiply to equal a certain number.

Vertex: The highest/lowest point of a parabola. (Dependent of the slope of the quadratic. Negative = Highest, Positive = Lowest)

Completing The Square: A method commonly used in mathematics to convert a quadratic to vertex form, find the roots of it, or make it be graphed easier.

Axes of Symmetry: An imaginary line that splits a parabola into two symmetrical pieces.

Formulas:

Standard Form: ax2 + bx + c

Factored Form: (a - r1)(a - r2)

Vertex Form: a(x - h)2 + k
