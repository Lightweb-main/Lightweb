Introduction

Polynomials can have several terms which are singular variables or numbers, just like the example above, but polynomials can also be just single numbers...

Polynomial Operations

Just like numbers and variables, polynomials have operations like addition, subtraction, multiplication, division, etc.

Adding Polynomials

Polynomials can be added with each other...

(3x3 - y2 + 5z) + (2x3 + 2y + 3)

To add polynomials, combine like terms raised to the same power...

(3x3 - y2 + 5z) + (2x3 + 2y2 + 3) = (3x3 + 2x3) + (-y2 + 2y2) + 5z + 3 = 5x3 + y2 + 5z + 3

And the sum of the polynomials is 5x3 - 3y2 + 5z + 3...

Adding Polynomials 1. Find like terms raised to the same power. 2. Combine like terms raised to the same power. 3. Ignore temrs without another like term raised to the same power. 4. Repeat steps 1 to 3 until all like terms are combined and or are simplified .

Subtracing Polynomials

Polynomials can also be subtracted with each other...

(2b5 - a3 - t2) - (5b5 - 3a3 + 3t2)

Do the same as Adding Polynomials but instead flip the signs of the second polynomials...

(2b5 - 5b5) + (a3 - (-3a3) + (t2 - 3t2) = -3b5 + 4a3 - 2t2

The difference is -3b5 + 4a3 - 2t2...

Subtracing Polynomials 1. Flip the signs of the second polynomial. 2. Combine like terms raised to the same power. 3. Ignore temrs without another like term raised to the same power. 4. Repeat steps 1 to 3 until all like terms are combined and or are simplified .
