Discriminants

Introduction

Discriminants in math when referring to the discriminant in polynomials is a value that is used to find the nature of the roots or solutions to polynomial like quadratics. Discriminants are shown as the uppercase greek letter, Delta or with a capital D...

Δ or D

The Formula for Discriminants of Quadratics

The formula used to find the discriminant in a quadratic is...

Δ = b2 - 4ac

This formula actually found within the quadratic formula...



x  =  -b  ±  b 2 - 4ac 2a

So for example, take this quadratic for example...

2x2 - 4x + 2 = 0

So from this quadratic we can see that...

a = 2 b = -4 c = 2 These are the coefficents

So from this we can plug in the values to the discriminant formula...

Δ = (-4)2 - 4(2)(2) Δ = 16 - 16 Δ = 0

The discriminant of this quadratic is zero.

The Formula for Discriminants of Cubics

The formula used to find the discriminant in a cubic is...

Δ3 = b2c2 - 4ac3 - 4b3d - 27a2d2 + 18abcd Also the discriminant of a cubic is shown as Delta with a small 3 next to it

So for example...

-x3 + 3x2 - 4x - 3 = 0

The Value Of Discriminants

When the discriminant of a polynomial is found, it tells the nature of the roots of the polynomial itself.

When Δ > 0

When the discriminant of a polynomial is greater than zero, that means that there are two real roots...

Finding Delta

x2 + 5x - 4 = 0 Δ = 52 - 4(-4) Δ = 25 + 16 Δ = 41

Finding The Roots



x  =  -5  ±  41 2

x  =  -5  +  41 2 ,    -5  -  41 2

When Δ = 0

When the discriminant of a polynomial is 0, that means that one of the roots are a repeated root, meaning that one of the roots is the same as another, this can eliminate into one root for a quadratic and two roots for a cubic...

Finding Delta

4x2 + 8x + 4 = 0 Δ = 82 - 4(4)(4) Δ = 64 - 64 Δ = 0

Finding The Roots



x  =  -8  ±  0 2

x  =  -8 2

x  =  -4

So the only root for this quadratic is -4.

When Δ < 0

When the discriminant of a polynomial is less than 0, that means that all of the roots are complex, or in a cubic that one of the roots are real and the other two are complex...

Finding Delta

2x2 + 3x + 5 = 0 Δ = 32 - 4(2)(5) Δ = 9 - 40 Δ = -31

Finding The Roots



x  =  -3  ±  -31 2

x  =  -3  ±  -31 2

x  =  -3  -  -31 2 ,   -3  -  -31 2

The roots of this quadratic are both complex.
