Completing The Square

Introduction

Completing the Sqaure is a common method used to solve quadratic equations and to derive the Quadratic Formula.

Steps to Complete the Sqaure

x2 + bx format

Example #1
To complete the sqaure on a quadratic equation with the format, "x2 + bx" without any c value, first look at the coefficent for bx in ax2 + bx...

x2 + 6x

So since the coefficent of bx in this quadratic is 6, we are going to use 6 for the next step. The next step is to divide the coefficent by 2...

6 / 2 = 3

Now that the coefficent is divided, now we sqaure it...

32 = 9

Now we add it and subtract it...

x2 + 6x + 9 - 9

Now we rewrite the ax2 + bx + c to (x - the halfed coefficent)2...

x2 + 6x + 9 - 9 ----> (x + 3)2 - 9

So when we complete the sqaure with x2 + 6x, we get (x + 3)2 - 9...

x2 + 6x ---> (x + 3)2 - 9 = 0

x2 - 15x

Now here's all the steps in order...

x2 - 15x

(-15/2)2 = (-15)2 / (2)2

x2 - 15x + 225/4 - 225/4

(x - 7.5)2 - 225/4

x2 - 15x ----> (x - 7.5)2 - 225/4 = 0

ax2 + bx + c format

For quadratics that have the format, "ax2 + bx + c", factor out a and complete the sqaure inside of the parantheses...
Example #1

3x2 - 5x + 4 = 0

Now since 3 is the coefficent of ax2, factor it out...

3(x2 - (5/3)x) + 4 = 0

Now complete the sqaure...

3(x2-(5/3)x)

(-5 / 3) / 2 = 5/6

(5/6)2 = 25/36

3(x2 - (5/3)x + 25/36 - 25/36) + 4 = 0

3((x - 5/6)2 - 25/36) + 4 = 0

Now distribute the 3 and simplify the equation...

3(x - 5/6)2 - 75/36 + 4 = 0

Since 4 = 144/36

3(x - 5/6)2

3x2 - 5x + 4 = 0 -----> 3(x - 5/6)2 + 69/36 = 0

Example #2
Here's another example...

2x2 - 8x + 4 = 0

Here are all the steps...

2x2 - 8x + 4 = 0

2(x2 - 4x) + 4 = 0

2(x2 -4x) + 4 = 0

-4 / 2 = -2

(-2)2 = 4

2(x2 - 4x + 4 - 4) + 4 = 0

2((x - 2)2 - 4) + 4 = 0

2((x - 2)2) - 8 + 4 = 0

2((x - 2)2) - 4 = 0

2x2 - 8x + 4 ----> 2(x - 2)2 - 4 = 0
