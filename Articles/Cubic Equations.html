Cubic Equations

Introduction

Cubic equations are polynomials that have one variable cubed (usually with a coefficient)...

ax³ + bx² + cx + d = 0

Similar to quadratics, a cannot equal zero in cubics…

2x² + 3x - 4 = 0

Since this function doesn’t have a variable with a power of 3 or in other words a = 0. This isn’t a cubic but rather a quadratic.

There are many ways to solve a cubic equation, but it depends on what the cubic equation is.

Factoring Back to Quadratics

Sometimes factoring a cubic is possible when d = 0…

3x³ - 4x² + 3x = 0

This cubic equation doesn’t have a fixed d value which means that one of the roots for this cubic is zero…

X = 0

We can also factor the cubic back to a quadratic…

x(3x² - 4x + 3)

Now we can solve for the other two roots by solving the quadratic in the parenthesis…

3x² - 4x + 3 = 0 4 ± √ (16 - 24) / 6
