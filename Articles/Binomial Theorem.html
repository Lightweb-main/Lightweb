Binomial Theorem

Introduction

The Binomial Theorem is a formula used to find the product of a binomial by itself...

(-4x + 2) * (-4x + 2) * (-4x + 2)

You see, we can't just multiply these together like numbers because every term would have to get multiplied by something. This is where the binomial theorem comes into play...



( a + b ) n = ∑ k = 0 n a n − k b k ⁢ ( n k )

This formula is called The Binomial Theorem and can be used to find the product of binomials as previously mentioned. Heres an example of the Binomial Theorem...

(a+b)0 = 1 (a+b)1 = a + b (a+b)2 = a2 + 2ab + b2 (a+b)3 = a3 + 3a2b + 3ab2 + b3

Notice how the exponents of the first a variable and the last b variable both are equal to the exponent of (a + b)...

(a+b)3 = a3... + ... b3

This is because of the formula...

This is the pattern that occurs when binomials are multiplied by themselves. The only thing missing are the coefficents for each varaible...

Pascal Triangle

If we look at the coefficents very closely, we can see that they form the Pascal Triangle...

This is very important since every individual coefficent is equal to each number on this triangle...

(a + b)0 = 1 (a + b)1 = 1a + 1b (a + b)2 = 1a2 + 2ab + 1b2 (a + b)3 = 1a3 + 3a2b + 3ab2 + 1b3 (a + b)4 = 1a4 + 4a2b + 6a2b2 + 4ab2 + 1b4

We can see that the coefficents of each "a" and "b" are equal to their respective number on the Pascal Triangle.

Binomial Coefficient Formula

But there is another formula of finding the Binomial Coefficent of a term...



( n k ) = n! k!(n - k)!

This formula is used to find Binomial Coefficients. The semantic (word) form of this is "N chooses K". So for example, lets solve the third number in the sixth row...



( 6 3 ) = 6! 3!(6 - 3)!

( 6 3 ) = 6! 3!3!

( 6 3 ) = 720 36

( 6 3 ) = 20

Lets see if that's correct...

Looks like the third number in the sixth row is 20, meaning that the formula is correct. (keep in mind that the 1 on the left counts as the '0th' number in the sixth row, thats why the third number in the sixth row equals 20)
