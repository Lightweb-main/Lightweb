Binomial Theorem

Introduction

The Binomial Theorem is a formula used to find the product of a binomial by itself...

(-4x + 2) * (-4x + 2) * (-4x + 2)

You see, we can't just multiply these together like numbers because every term would have to get multiplied by something. This is where the binomial theorem comes into play... ( a + b ) n = ∑ k = 0 n a n − k b k ⁢ ( n k ) This formula is called The Binomial Theorem and can be used to find the product of binomials as previously mentioned. Heres an example of the Binomial Theorem...

(a+b)0 = 1

(a+b)1 = a + b

(a+b)2 = a2 + 2ab + b2

(a+b)3 = a3 + 3a2b + 3ab2 + b3

Notice how the exponents of the first a variable and the last b variable both are equal to the exponent of (a + b)...

(a+b)3 = a3... + ... b3

This is because of the formula... ( a + b ) 3 = ∑ k = 0 n a n − k b k ⁢ =

= a3b0 + + +

= a3 + a2b1 + +

= a3 + a2b + a1b2 +

= a3 + a2b + ab2 + a0b3

= a3 + a2b + ab2 + b3

This is the pattern that occurs when binomials are multiplied by themselves. The only thing missing are the coefficents for each varaible...

= a3 + 3a2b + 3ab2 + b3

If we look at the coefficents very closely, we can see that they form the Pascal Triangle... This is very important since every individual coefficent is equal to each number on this triangle...

(a + b)0 = 1

(a + b)1 = 1a + 1b

(a + b)2 = 1a2 + 2ab + 1b2

(a + b)3 = 1a3 + 3a2b + 3ab2 + 1b3

(a + b)4 = 1a4 + 4a2b + 6a2b2 + 4ab2 + 1b4

We can see that the coefficents of each "a" and "b" are equal to their respective number on the Pascal Triangle.

( 6 3 ) = 6! 3!(6 - 3)!
