Absolute Value Equations

Introduction

Absolute equations are equations that contain the absolute value of a number (s) like this one…

|2x - 4| = 18

This example is a multi-step absolute equation. To solve an absolute equation, take a look at it. In this equation the absolute value of 2x - 4 equals 18. In an absolute equation, there are two solutions. Like for the example above, we can turn it into two separate equations…

|2x - 4| = 18

We can simply remove the vertical lines and solve…

2x - 4 = 18 2x = 22 x = 11

Or we can remove the vertical lines, make the sum negative and solve…

2x - 4 = -18 2x = -14 x = -7

But what if two absolute values are equal to each other…

|2x - 4| = |4x + 2|

We can remove the two pairs of vertical lines…

2x - 4 = 4x + 2 2x = 4x + 6 -2x = 6 x = -3

And for the second solution, we would remove the first pair of vertical lines on the left and make the equation on the right negative…

2x - 4 = -4x - 2 2x = -4x + 2 6x = 2 x = 2/6 = 1/3

The two solutions for this equation are ⅓ and 1.

What if there is multiplication…

3|x - 2| = 26

Divide both sides by 3…

|x - 2| = 26/3

We can find out the two solutions…

x - 2 = 26/3 x - 2 = -26/3 x = 32/3 x = -20/3

But what if there are two absolute values on one side…

|3x + 3| + |-4x - 2| = 24

We can move one of the absolute values over to the other side…

-4x - 2 = 24 + 3x + 3 -4x - 2 = 3x + 27 -7x - 2 = 27 -7x = 29 x = -29/7 3x + 3 = 24 - 4x - 2 3x = 21 - 4x - 2 3x = 19 - 4x 7x = 19 x = 19/7 x = -29/7, 19/7

But what if the question is an inequality with an absolute value…

|x + 2| < 16

We can remove the vertical lines and keep the sign the same…

x + 2 < 16 x < 14

And we can remove the vertical lines, switch the sign, and switch the numbers sign…

x + 2 > -16 x > -18

The final answer is…

-18 < x < 14

What about the sign pointing the other way…

|3x - 5| > 16

Remove the vertical signs and solve…

3x - 5 > 16 3x > 21 x > 7

And do the same for the next but switch the sign around and the numbers sign …

3x - 5 < -16 3x < -11 x < -11/3

Unlike absolute equations where the sign is like this “<” instead of this “>”, x could either be smaller than -11/3 or bigger than 7, it’s not in between…

7 < x < -11/3 INVALID EQUATION x > 7 or x < -11/3

But what if there are two absolute values and the question is an inequality…

|5x - 7| + |3x + 2| < 18

Turn the less than sign into an equal sign…

|5x - 7| + |3x + 2| = 18 Solve normally…

Move one of the values to the other side and switch its signs.

3x + 2 = 18 - 5x + 7 3x = 18 - 5x + 5 8x = 23 x = 23/8

Move the other value to the other side and switch its signs.

3x + 2 = 5x - 7 + 18 3x + 2 = 5x + 11 3x + 2 = -5x - 11 3x = -5x - 13 8x = -13 x = -13/8

Put the smaller number first…

-13/8

The smallest number goes last…

-13/8 > x > 23/8

Why do Absolute Equations have 2 Solutions?

Absolute Equations have two solutions because absolute values are a distance metric, meaning that both a positive and negative value both result in a positive, absolute value…

|x| = |-x|

This applies to multi-step absolute equations where two solutions can be found.
