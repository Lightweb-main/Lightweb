Linear Functions

Introduction

Linear functions are linear equations that represent a straight line on a graph like this example…

This line has a slope of one since it always moves up and to the right 1. Linear functions sometimes aren’t equal throughout the graph, sometimes the x value could change differently from the y value.


Examples of linear functions are these…

y = 2x + 3 y = -4x + 5 y = 1/2x - 4 y = -1x + 3 y = -1/4x - 1

Linear functions are commonly written as a table…

X Y
0 0
1 1
2 2
3 3
4 4

They can also be written as a slope equation as the examples that are above this text…

y = 3x - 1

Sometimes, linear functions are written like this…

f(x) = 2x - 4

Linear Functions always have four types that they can land in. They can either be a positive slope, a negative slope, a zero slope, or an undefined slope.

Positive Slopes

A positive slope is a line that goes through the left, bottom side to the top, right side like this…

Since this line goes from the bottom to the top when you look at it from left to right, it is a positive slope.



Negative Slopes

A negative slope is a slope that goes from the opposite side, from the right into the left like this…

This is a negative slope because the line starts at the top, left side while it goes through the origin into the bottom, right side. You can see that both slopes are different, they are mirrored to each other.



Zero Slopes

A zero slope is a slope that has a fixed y value, this means that it is just a flat line like this…

This is a zero slope because the y value doesn’t change at all throughout the graph, this is considered a zero slope line because no difference in y is found which basically translates to a zero over a number as a fraction…

Slope = 0/x

Undefined Slopes

Lastly, an undefined slope is a slope that instead has a fixed x value like this…

This line has an undefined slope because the difference in x is 0 and a number divided by 0 is undefined, which means that the slope is undefined…

Slope = x/0 = undefined

The four types of slopes are examples of Slope-Intercept linear functions.



Linear functions in their standard form are…

Ax + By = C

Examples include…

2x + 3y = 6 x + 4y = 8 3x + 1y = 6 6x + 1y = 12

These can be rewritten as slope intercept linear functions. First we would have to get y on it’s own while x is on the other side. We can do this by adding/subtracting the opposite on both sides…

6x + 1y = 12 y = -6x + 12

This slope would be negative and the slope would be -6x. This is the process of converting a standard form linear function to a slope intercept form function.



Linear functions can also be written as another form, such as the Point-Slope form…

y - y1 = m(x - x1)

This may look like the most complicated formula but it actually is identical to other forms of linear functions. First, there are small numbers beside some of the variables. These numbers are there to indicate that they are different from the variables that don’t have them. This is similar to the slope formula which also has these small numbers…

y2 - y1 x2 - x1

The Point-Slope form is actually the exact same from this. By looking at the formula, the two y’s are the y values of two dots, x being the same thing except for the x values of both dots, but m is the slope. So the two x, times the slope equals the two y values. Let's say we had two dots on a coordinate graph...

We can see that one of the dots is placed at, (-2, 0) while the other is placed at (0, 3). We can fill in the values in the formula…

0 - (-3) = m(0 - (-2)) 3 = m(2) 3 = m * 2 3 = 2m

Now the slope times 2 equals 3. We can figure out the slope. We can divide both sides by 2…

1.5 = m
