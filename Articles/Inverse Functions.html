Inverse Functions

Introduction

Inverse functions are functions that reverse or do the opposite things as their normal or regular function counterparts. For example, the conversion from celsius to fahrenheit is...

f(x) = (9/5 * x) + 32

This is the formula used to convert celsius temperatures into fahrenheit temperatures, however if we do the opposite and convert fahrenheit temperatures to celsius, then the formula would be...

c(x) = (x - 32) * 5/9

Notice how every operation on x is reversed, the +32 turned into -32 and 9/5 turned into 5/9.

Inverse Function Notation

Inverse functions are shown as the regular function but instead with a -1 as the exponent and the input (x) value being the output (y)...

f-1(y) = f(x)

This is similar to whole numbers with negative exponents...

2-1 = 1/21

f(x) = 3x - 2

f-1(y) = (x + 2) / 3

Finding Inverse Functions

To find inverse functions, reverse every operation in the function, so for example addition becomes subtraction, multiplication becomes division, and paratheses are split...

f(x) = √x

f-1(y) = x2

g(x) = 7x2

g-1(y) = (√x / 7), -(√x / 7)
