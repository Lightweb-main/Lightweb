Inverse Functions

Introduction

Inverse functions are functions that reverse or do the opposite things as their normal or regular function counterparts. For example, the conversion from celsius to fahrenheit is...

f(x) = (9/5 * x) + 32

This is the formula used to convert celsius temperatures into fahrenheit temperatures, however if we do the opposite and convert fahrenheit temperatures to celsius, then the formula would be...

c(x) = (x - 32) * 5/9

Notice how every operation on x is reversed, the +32 turned into -32 and 9/5 turned into 5/9.

Inverse Function Notation

Inverse functions are shown as the regular function but instead with a -1 as the exponent and the input (x) value being the output (y)...

f-1(y) = f(x)

This is similar to whole numbers with negative exponents...

2-1 = 1/21



f(x) = 3x - 2

f-1(x) = (x + 2) / 3

Finding Inverse Functions

To find inverse functions, first change the f(x) in the equation to y...

f(x) = 9x - 5

y = 9x - 5

Next, swap y and x...

x = 9y - 5

Then solve the equation...

Adding 5 on both sides

x + 5 = 9y

Dividing both sides by 9

x + 5 9  =  y Now change the x in the equation with f -1(x)... f -1 (x)  =  x + 5 9

Verifying Inverse Functions Through Composition

You can verify if two functions are the inverse of each other through composition. In order for two functions to be the inverse of each other, they need to meet these equations...

f(f -1(x)) = x

f -1(f(x)) = x

So if f(x) composed of f-1(x) or if f-1(x) composed of f(x) equals x, then that means that the two functions are inverses of each other. For example...

f(x) = -5x + 3

g(x)  =  x - 3 -5 So to see if they're inverses of each other, we must plug in the equation in f(x) over to g(x) and vice versa...

f(g(x))

f(g(x)) = -5((x - 3) / -5) + 3 =

x - 3 + 3

f(g(x) = x



g(f(x)

g(f(x)  =  (-5x + 3) - 3 -5

g(f(x)  =  -5x -5

g(f(x)  =  -5x -5  =  x

g(f(x)) = x

f(g(x)) = x

Since f(g(x)) and g(f(x)) equal x, that means that the two functions are inverses of each other.
