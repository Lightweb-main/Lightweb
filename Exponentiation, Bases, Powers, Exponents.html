Exponentiation, Powers, Bases, and Exponents

Introduction

Exponents are the third hyperoperation in mathematics which is known for having a number multiplied by itself a numerous amount of times…

2 3

The bigger number in exponentiation is called the base number…

2 3

The smaller number to the upper right side in exponentiation is called the exponent…

2 3

Both of them together are called a power…

2 3

Solving Exponents

Now exponents can reach big numbers much quicker than multiplication does, for example…

4 3

We can see that the base is 4 and the exponent is 3. One method is to convert the two numbers to multiplication (However the exponent may be too large to make the power to be rewritten to multiplication). This is telling us that the value of 4 to the power of 3 is just 4 * 4 * 4, or 4 times itself three times…

4 3 = 4 * 4 * 4

Since this is now multiplication, it is easier to solve…

4 3 = 4 * 4 * 4 = 16 * 4 = 64

So 4 to the power of 3 is 64…

4 3 = 64

Another example would be 3 to the power of 7…

3 7

Since the exponent is small enough, multiplication can be used…

3 7 = 3 * 3 * 3 * 3 * 3 * 3 * 3 = 9 * 3 * 3 * 3 * 3 * 3 = 27 * 3 * 3 * 3 * 3 = 81 * 3 * 3 * 3 = 243 * 3 * 3 = 729 * 3 = 2187

3 to the power of 7 is 2187.

Multiplying and Dividing Powers

Powers can be multiplied by other powers like this one…

x 3 * x 2 = ???

The rule to multiplying two powers together is that the product is the base on both multipliers with both exponents adding together…

x a * x b = x a + b

So in this example we just add two of the exponents, 3 and 2…

x 3 * x 2 = x 5

The product of x3 plus x2 is x5.

Multiplying and Dividing Powers (Different Bases)

Sometimes the exponents can be the same but the bases aren’t…

5 12 * 3 12 = ???

Instead of keeping the bases and adding the exponents, we multiply the bases and keep the exponents…

5 12 * 3 12 = ( 5 * 3 ) 12 = 15 12

The answer is 15 to the power of 12.

Multiplying and Dividing Powers (Different Bases & Exponents)

If both the bases and the exponents are different from each other, find the answers to both powers then multiply…

3 3 + 2 5 = ??? 3 3 = 3 * 3 * 3 = 27 2 5 = 2 * 2 * 2 * 2 * 2 = 32 3 3 + 2 5 = 27 + 32 = 59 3 3 + 2 5 = 59

If you are dividing exponents instead of multiplying, you do the opposite and instead subtract the exponents…

x a x b = x a - b

Multiplying Powers With Multiple Numbers/Variables

Sometimes a math problem with exponentiation can have multiple numbers and variables like this one…

2 3 x 2 y 7 ( 2 5 x 3 y 2 ) = ???

This can be solved by adding the exponents on every power…

2 3 x 2 y 7 ( 2 5 x 3 y 2 ) = 2 3 + 5 x 2 + 3 y 7 + 2 = 28 x 5 y 9 = 256 x 5 y 9

If you are simplifying the equation, it is 28x5y9. If you are evaluating the equation, it is 256x5y9, both are equal to each other since 28 = 256.

Raising a Power to another Power

Raising a power to another power is basically exponentiation with a power…

( 2 3 ) 5

To find the answer, multiply the exponents together…

( x a ) b = x a + b ( 2 3 ) 5 = 2 15

So 2 to the power of 3 to the power of 5 equals 2 to the power of 5.

With one number/variable in the brackets (excluding the exponent), this is simple but things change with two or more numbers in the brackets like this one…

( 2 4 x 3 y 2 ) 5

Do the same thing with multiplying every exponent in the brackets by 5…

( 2 4 x 3 y 2 ) 5 = 2 20 x 15 y 10

Raising a Power to many other Powers

This section of this article is basically raising a power to a power to a power, basically repeated exponentiation…

( ( 3 x 4 y 3 ) 3 ) 6

Quick note before solving this is that with parentheses…

3 3 3 ≠ ( 3 3 ) 3

The parentheses matter a lot because in PEMDAS, parentheses go first which means that 3 to the power of 3 on the right side goes first…

3 3 3 ≠ ( 27 ) 3

With the equation on the left, since there aren't any parentheses, meaning that the first, larger exponent gets multiplied by itself…

3 27 ≠ ( 27 ) 3

Since 3 to the power of 27 is much, much larger than 27 to the power of 3, they aren’t equal. Back to the problem above…

( ( 3 x 4 y 3 ) 3 ) 6

First we multiply the exponents by three…

( 3 3 x 12 y 9 ) 6

Then we multiply the exponents by six…

3 18 x 72 y 54

So the answer is 3 to the power of 18 times x to the power of 72 times y to the power of 54…

( ( 3 x 4 y 3 ) 3 ) 6 = ( 3 18 x 72 y 54 )
