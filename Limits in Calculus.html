Limits in Calculus

Introduction

A limit in calculus describes a specific value (y) that a function approches when it's input (x) approches it. Limits are shown as...

lim x → a f ( x ) = L

Limits are shown in mathematics that some functions that approach a certain value tend to get closer and closer to L or limiting value, the output of the limit. But see what happens when the value "1" is plugged into this function...

lim g → 2 ( 3 g 3 - 24 ) ( g 2 - 4 ) ( 3 ( 2 ) 3 - 24 ) ( ( 2 ) 2 - 4 ) → ( 24 - 24 ) ( 4 - 4 ) → 0 0

When 2 is the input of the function, the limiting value becomes undefined. So let's do something that is close to 2 but isn’t exactly 2…

let  g = 1.9 ( 3 ( 1.9 ) 3 - 24 ) ( ( 1.9 ) 2 - 4 ) → ( 20.577 - 24 ) ( 3.61 - 4 ) → -3.423 -0.39 = 8.78

When g = 1.9, the function outputs 8.78, which can be rounded to 9. In a limit, no matter how many digits the variable approaching a certain number has without exactly being that number, it will get close to L but not exactly L…

let  g = 1.99 ( 3 ( 1.99 ) 3 - 24 ) ( ( 1.99 ) 2 - 4 ) → ( 23.642 - 24 ) ( 3.9601 - 4 ) → -0.358 -0.0399 = 8.972 let  g = 1.999 ( 3 ( 1.999 ) 3 - 24 ) ( ( 1.999 ) 2 - 4 ) → ( 23.96402 - 24 ) ( 3.996 - 4 ) → -0.03598 -0.003999 = 8.9972

On a graph, this limit would show up like this…

If you look at when x = 2, the line cuts off, leaving an empty space. This happened because we don’t really know the value of x when it equals 2 because the function itself outputs 0 over 0. But when applied in real life, this logic wouldn’t make any sense, a slope suddenly cuts off and magically there is nothing there. This is where we would go above 2 (going from the opposite side) and try to get as close to 2…

let  g = 2.1 ( 3 ( 2.1 ) 3 - 24 ) ( ( 2.1 ) 2 - 4 ) → ( 27.783 - 24 ) ( 4.41 - 4 ) → 3.782 0.41 = 9.224 let  g = 2.01 ( 3 ( 2.01 ) 3 - 24 ) ( ( 2.01 ) 2 - 4 ) → ( 24.361 - 24 ) ( 4.0401 - 4 ) → 0.361 0.0401 = 9.0025

As we keep adding more zeros, the closer the quotient gets to 2. This means that when x is smaller or bigger than 1, the result will always be close to 2 but not exactly 2 which is okay.

Finding the Limiting Value

The limiting value is the number that is equal to the limit like…

lim r → 3 r³ + 3 2r = ??

To find the limiting value, solve the function first…

lim r → 3 r² + 3 2r = r² 2  +  3 2r

Now substitute r as 3…

lim r → 3 = 3² 2  +  3 2 * 3   =   9 2  +  3 6  =  4.5 + 0.5  = 5.0

The limiting value is 5. You can also directly substitute r as 3…

lim r → 3 = r³ + 3 2r  =  27 + 3 6   =   30 6  =  5

We get the same answer from this.
