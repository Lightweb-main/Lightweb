Here are words that are often used within HTML

There are several words that will come up during this course other than just HTML. Lets start with tags. Tags are the building blocks of HTML. They appear as text inside < angle brackets > just like this example...

< tag >

Everything in HTML has a starting tag to intiate it...

< startingtag >

Most tags have an ending tag...

< starting tag >< ending tag >

All ending tags have one thing distinctively different from starting tags. That difference is that every ending tag has a slash / between the text and the bracket...

< example >< /example >

Over this course, you will see a lot of different tags being used. And you will see this next term commonly also.

Elements are defined as the group containing a starting tag, text content, and an ending tag like this...

< exampleElement >Text Content< /exampleElement >

Another word that you will see alot is style...

< example style="text-align: center;"> < /example>

Basically "style" or "styling" an element basically means that you are changing a property of an element like aligning the text content to align in the center just like the example above.
