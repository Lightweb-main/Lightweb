HTML Paragraphs

Up next are paragraphs. An html paragraph is declared using the < p > tag and concludes with the < /p > tag. Paragraphs in HTML function similar to just plain text but there are some key differences between the two...

Paragraphs Plain Text
Start on a new Line. Piles up on one line, even when text is on a different line.
Can be styled directly. Cannot be styled directly.
Paragraphs are commonly used alongside headings to generate organized pieces of information.
