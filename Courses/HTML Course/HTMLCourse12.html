HTML Comments

You can add, non-operative pieces of text called comments in your program. Comments can explain or summarize how the program works...

< !-- This is a comment that doesn't affect the output of the program! -- >

Comments can be used to disable code..

< !-- < p > This paragraph doesn't work because it's in a comment < /p > -->

Comments can also span across multiple lines like this...

< !-- all of
this is
a comment -->
