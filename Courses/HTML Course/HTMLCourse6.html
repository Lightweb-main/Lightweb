The Structure of an HMTL Website

Every website will certainly have a unique structure and will use different elements at different times, but these tags are always used in an html website...

< body >

< head >

< html > < / html >

< !DOCTYPE html >

The Start

Right after the document type has been declared, < html > is written, this is to intitate html code...

< html >

< / html >

After the < html > tag, two parts of the site are declared, these parts are known as < body > and < head > and both have ending tags to close them off...

< html >

< head >

< /head >

< body >

< /body >

< / html >

The < body > tag of the website is where the contents of it are stored. The < head > tag is where the styling, language, and encoding are kept.
