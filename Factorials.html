Factorials

Introduction

Factorials are the products of an integer and all the integers under it. Examples of factorials would be…

4 ! 0 ! 15 !

Every Factorial has an exclamation mark to the right of it. The formula for finding factorials is…

n ! = n * ( n - 1 ) * ( n - 2 ) * ... * 1

By using this formula, we can find the value of any factorial. For example... 4 ! = 4 * 3 * 2 * 1 = 4 * 3 * 2 = 12 * 2 = 24

By using formulas for factorials, the factorial of 4 is 24.
